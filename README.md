# Flutter Project (tested on android)

### Features: 
1. Adding employee;
2. Adding employee children;
3. Saving data using sqlite;
3. Deleting data;

### How to begin
1. ```git clone https://github.com/Goolpe/ekf.git```
2. ```cd ./ekf```
2. ```flutter pub get```

### Debug
```flutter run```
