# Flutter Project List (tested on android)

### Features: 
1. Add employee;
2. Add employee's children;
3. Store data in sqlite;
3. Delete data;

### How to begin
1. ```git clone https://github.com/Goolpe/ekf.git```
2. ```cd ./ekf```
2. ```flutter pub get```

### Debug
```flutter run```
